# Projeto-Consumo-de-API
Projeto de consumo de API, a API escolhida foi a Dog CEO API. 

Este projeto consiste no desenvolvimento de uma aplicação web responsiva utilizando HTML, CSS, JavaScript e PHP, com o objetivo de consumir uma API pública. A aplicação realiza requisições HTTP para a API Dog CEO, processa os dados em formato JSON no back-end e os exibe de forma organizada no front-end. O usuário pode interagir por meio de botões e campo de busca, visualizando imagens de cães e informações associadas, com tratamento de erros para garantir uma melhor experiência.
